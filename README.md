Magma and PARI/GP code to accompany the paper

 ``p-adic Elliptic Polylogarithms and Cubic Chabauty''  (https://arxiv.org/abs/2604.20662)       

 by Jennifer S. Balakrishnan, Francesca Bianchi, Netan Dogra


The examples in Section 6 of the paper are produced using the files `rank0ex.m`, `rank1ex.m`, `rank2ex389a1.m`, `rank2ex433a1.m`.

Notes:
- Built on Magma V2.28-7  and PARI/GP Version 2.15.5.
- Certain Magma files build on code written with Jan Tuitman, Steffen  Müller, Jan Vonk and are indicated as such.
- The PARI/GP Jupyter notebook includes the "elldilog" function of Fernando Rodriguez Villegas, which was updated for compatibility with the version of PARI/GP above, using GPT 5.2.
